Forge Installation Guide

  1. Install Java 17 (if needed): Minecraft 1.20.1 and Forge require Java 17. If you don't have it installed, download it from: Oracle Java SE Development Kit 17. Choose the appropriate installer for your operating system.
  2. Download Forge: Visit the official Forge website: Minecraft Forge 1.20.1
  3. Click on the Installer link to download the Forge installer.
  4. Run the Installer: Double-click the downloaded Forge installer JAR file to run it.
  5. Select "Install Client": In the installer window, make sure the "Install client" option is selected.
  6. Choose your Minecraft installation directory (it should be automatically detected).
  7. Click OK to start the installation.
  8. Open the Minecraft Launcher: Launch the Minecraft Launcher.
  9. Select the Forge Profile: In the launcher, you should now see a new profile named "forge-1.20.1" (or similar). Select this profile before launching the game. If you don't see it, ensure "Versions" is selected in the bottom left, and look for the Forge installation.
  10. Install the Modpack: Download the QuibusCraft modpack from the provided link (above). The modpack files should be placed in the mods folder inside your Minecraft installation directory. For example, if your Minecraft installation is in `C:\Users\YourName\AppData\Roaming\.minecraft`, you should place the modpack files in `C:\Users\YourName\AppData\Roaming\.minecraft\mods`.
  11. Launch Minecraft: Launch the game with the Forge profile selected. The modpack should now be loaded.

Note: Make sure you have Minecraft version 1.20.1 installed before installing Forge. You may need to run Minecraft 1.20.1 once before Forge can be installed.
